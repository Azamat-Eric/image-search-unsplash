# 🖼️ EASY_05 — Image Search (Unsplash)

Поиск картинок высокого качества из Unsplash. Бесконечная прокрутка, скачивание, поиск по жанрам.

## 🚀 Запуск

```bash
cd EASY_05_Image_Search
npm install
npm start
```

Открой **http://localhost:3005**

## ✨ Что делает

- Поиск картинок по запросу (на любом языке)
- Готовые жанры (природа, бизнес, еда, технологии)
- Бесконечная прокрутка (показывает по 20)
- Скачивание в один клик
- Превью с автором и ссылкой

## 💼 Бизнес-применение

- **Stock-photo маркетплейс** — добавь свои фотки рядом, бери комиссию
- **Для дизайнеров и SMM** — premium-плагин для Photoshop/Figma
- **Контент-ассистент** — посты для соцсетей с готовыми картинками
